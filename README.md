DripCase: Phone Case Store
A final group project for the University of Technology. This is a simple, responsive e-commerce website for browsing and purchasing phone cases.
 
 Features
•	Homepage: Features a landing section with featured products and promotions.
•	Product Catalog: Browse all available phone cases in a grid view.
•	Product Detail Page: View detailed information, images, and prices for a specific case.
•	Shopping Cart: Add items to the cart, update quantities, and see a running total.
•	User Accounts: Users can register, log in, and view their order history.
•	Admin Dashboard: Administrators can view user analytics and sales invoices.
•	Responsive Design: The layout adapts to different screen sizes for an optimal viewing experience.
Technologies Used
This project was built using the following tools and frameworks:
•	HTML5: For the structure and content of the web pages.
•	CSS3: For styling and layout, including Flexbox and/or Grid.
•	JavaScript (ES6+): For interactive functionality, DOM manipulation, and cart logic.
•	Font Awesome: For icons.
